# 3d-Game-Creator
A Claude Cod skill that turns making 3D games from a back-and-forth conversation to a simple 3-4 sentence prompt.
# Install Instructions:
Go to Claude code, press customize, skills and then the plus button and upload the .skill file from here.


# Usage Instructions:
Ask Claude to make a 3D Game and specify the JS 3D lib (options listed below).
1st. Three.js (lightweight and flexible)
2nd. Babylon.js (Better graphics but more token usage and space)
That's it. Hope you have fun!
